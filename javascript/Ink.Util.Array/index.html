.groupBy(arr, [options]) method

Loops through an array, grouping similar items together.

Accepts

  • arr

    {Array} The input array.

  • options

    {Object} options object, containing:

  • options.key

    {Function} A function which computes the group key by which the items are grouped.

  • options.pairs

    {Boolean} Set to `true` if you want to output an array of `[key, [group...]]` pairs instead of an array of groups.

Returns

  • An array of arrays of chunks.

Example

{% highlight html %}{% raw %} InkArray.groupBy([1, 1, 2, 2, 3, 1]) // -> [ [1, 1], [2, 2], [3], [1] ] InkArray.groupBy([1.1, 1.2, 2.1], { key: Math.floor }) // -> [ [1.1, 1.2], [2.1] ] InkArray.groupBy([1.1, 1.2, 2.1], { key: Math.floor, pairs: true }) // -> [ [1, [1.1, 1.2]], [2, [2.1]] ] {% endraw %}{% endhighlight %}

.range(start, stop, [step]) method

Simulates python's range(start, stop, step) function.

Creates a list with numbers counting from start until stop, using a for loop.
.
The optional step argument defines how to step ahead. You can pass a negative number to count backwards (see the examples below).

Accepts

  • start

    The array's first element.

  • stop

    Stop counting before this number.

  • step

    1Interval between numbers. You can use a negative number to count backwards.

.reduce(array, callback, initial) method

Replacement for Array.prototype.reduce.

Produces a single result from a list of values by calling an "aggregator" function.


Falls back to Array.prototype.reduce if available.

Accepts

  • array

    {Array} Input array to be reduced.

  • callback

    {Function} `function (previousValue, currentValue, index, all) { return {Mixed} }` to execute for each value.

  • initial

    {Mixed} Object used as the first argument to the first call of `callback`

Example

{% highlight html %}{% raw %} var sum = InkArray.reduce([1, 2, 3], function (a, b) { return a + b; }); // -> 6 {% endraw %}{% endhighlight %}
