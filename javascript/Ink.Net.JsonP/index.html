Ink.Net.JsonP class

Executes a JSONP request

Methods
Method name Description
new JsonP(uri, options) Constructor

new Ink.Net.JsonP(uri, options)

Accepts

  • uri

    Request URL

  • options

    Request options

  • options.onSuccess

    Success callback

  • options.onFailure

    Failure callback

  • options.failureObj

    Object to be passed as argument to failure callback

  • options.timeout

    Timeout for request fail, in seconds. defaults to 10

  • options.params

    Object with the parameters and respective values to unfold

  • options.callbackParam

    Parameter to use as callback. defaults to 'jsoncallback'

  • options.internalCallback

    Name of the callback function stored in the Ink.Net.JsonP object.

  • options.randVar

    (Advanced, not recommended unless you know what you're doing) A string to append to the callback name. By default, generate a random number. Use an empty string if you already passed the correct name in the internalCallback option.
